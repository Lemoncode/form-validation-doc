# Fonk

Fonk is a pure javascript form validation library.

It allows you to:

- Define validations that apply to your form in a declarative way.
- Isolate your form validation business logic from the ui.
- Unit test your form validation without having to mount the ui.
- Create reusable validation rules that can be easily unit tested.

[Try it out](https://github.com/Lemoncode/fonk/tree/master/examples/vanilla/js)

Fonk is framework extension, and can be easily plugged into many libraries / frameworks,
in this documentation you will find integrations with:

- Plain Vanilla ES6

- React.

- React Final Forms.

- Formik.

- Vuejs.
